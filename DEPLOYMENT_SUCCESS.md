### Issue: Images Don't Upload
**Possible causes**:
1. R2 bucket doesn't exist ‚Üí Check Cloudflare R2
2. R2 credentials wrong ‚Üí Verify in Vercel dashboard
3. CORS issue ‚Üí Check R2 bucket settings

**Check**: Upload a test image and watch browser console

### Issue: Database Errors
**Possible causes**:
1. RLS policies too restrictive
2. Tables missing
3. Permissions not set

**Fix**: Run [VERIFY_DATABASE_SETUP.sql](VERIFY_DATABASE_SETUP.sql) in Supabase

**Fix**: Add when ready:
